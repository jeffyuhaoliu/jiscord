Here's a summary of everything implemented:

---

**jiscord-eul.2 — Login + Register pages**
- `src/context/AuthContext.tsx` — `AuthProvider` with `{user, token, login, logout}`; JWT stored in memory ref (not localStorage — XSS-safe)
- `src/pages/LoginPage.tsx` — email/password form → `POST /api/auth/login` → redirects to `/channels`
- `src/pages/RegisterPage.tsx` — username/email/password → `POST /api/auth/register` → auto-logs in → redirects

**jiscord-eul.3 — useGateway WebSocket hook**
- Already implemented; added `TYPING_START` to `EventMap` with typed `TypingStartPayload`

**jiscord-eul.4 — Guild sidebar + Channel list**
- `src/components/GuildSidebar/` — fetches `GET /api/data/guilds/me` (passes `X-User-ID` header), renders guild icons
- `src/components/ChannelList/` — fetches `GET /api/data/guilds/:id/channels`, renders `#channel-name` list
- `data-service/src/index.ts` — added `GET /guilds/me` and `GET /guilds/:guildId/channels` endpoints

**jiscord-eul.6 — MessageInput component**
- `src/components/MessageInput/` — textarea, Enter to send (Shift+Enter = newline), throttled `TYPING_START` emission, live typing indicator

**Updated wiring**

- `main.tsx` — wrapped with `<BrowserRouter>` + `<AuthProvider>`
- `src/pages/ChannelsPage.tsx` — full layout: GuildSidebar + ChannelList + MessageList + MessageInput
